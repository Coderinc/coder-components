# Coder-Components
###### Small library of repeatable components used on the Coder Platform

### Component List
##### Elements
* Avatar
* Button
* Chip
* Container
* H1
* H2
* H3
* H4
* IconChip
* Paragraph
* ProgressBar
* Tile
* Wrapper
##### Components
* AnimatedNumber
* PhaseBar
* RequirementBar
* RangeSlider
* Slider

### Usage
```bash
npm install coder-components
```

```javascript
import { Button } from 'coder-components';
// ...
import 'coder-components/styles/css/styles.css';
```-
